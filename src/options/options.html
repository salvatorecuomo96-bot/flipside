FlipSide

FlipSide works out of the box with no setup. Optionally add your own API key for a dedicated personal quota.

When on, FlipSide silently analyzes each article you open and sets the badge before you click — using your key on every page load. Turn off to only use your key when you click the button.

Privacy

On article pages, FlipSide automatically sends the title, URL, and text to the analysis service to set the badge. Clicking the button opens the result panel — if already analyzed, nothing extra is sent. Non-article pages are never analyzed. No account is required and no personal data is collected.

Your API key is stored only on this device using chrome.storage.local. It is never sent to FlipSide's servers or shared with anyone. When you use your own key, your article data goes directly to the provider you selected — bypassing the shared proxy entirely. You can remove it at any time with the Clear key button above.
