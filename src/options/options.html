When on, FlipSide silently analyzes each article you open and sets the badge before you click — using your key on every page load. Turn off to only use your key when you click the button.

Privacy

On article pages, FlipSide automatically sends the title, URL, and text to the analysis service to set the badge — no click needed. Clicking the button opens the result panel (no extra data sent if already analyzed). Non-article pages are never analyzed. Your optional API key is stored only on this device (chrome.storage.local) and sent only to Groq — never to FlipSide's proxy.
